# Backbase Launchpad-CLI tools

Backbase Launchpad building for widgets / modules 

# Information
| name                  | version       | node      |
| ----------------------|:-------------:| ---------:|
| bb-lp-cli             | 0.0.3         | >= 0.10   |


## Install

```
npm i bb-lp-cli -g 
```

## Usage

Using **bblp** as binary.

```bash
bblp build
```
