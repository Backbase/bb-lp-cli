# Backbase Launchpad-CLI tools

Backbase Launchpad tools for widgets / modules 

# Information
| name                  | version       | node      |
| ----------------------|:-------------:| ---------:|
| bb-lp-cli             | 0.0.2         | >= 0.10   |


## Requirements
- git
- nodejs
- bower
- curl

## Install

```
npm i bb-lp-cli -g 
```

## Usage

Using **bblp** as binary.

Stand alone development
### Start local server: http://localhost:3000

arguments:

- **NONE**

options:

- **NONE**
    
```bash
bblp start
```

### Build: 

arguments:
    
- **NONE**

options:

- **NONE**

```bash
bblp build
```
