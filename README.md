# BackBase-CLI 

Backbase tool for widgets / modules



#Information
| name                  | version       | node      |
| ----------------------|:-------------:| ----------:|
| backbase              | 0.0.1         | >= 0.10    |

## Usage

## Requirements
- git
- nodejs
- bower

## Install

```
npm i backbase -g 
```

## Commands

Generate a new widget from template:

```bash
backbase generate <url>
```
